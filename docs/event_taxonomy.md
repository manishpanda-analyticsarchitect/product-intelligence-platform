# Product Event Taxonomy

## What is an Event?
An event represents a single action performed by a user in the product.
Each event is tied to a user, an account, a feature, and a timestamp.

---

## Core Event Categories

### 1. Authentication Events
- user_login
- user_logout

### 2. Navigation Events
- dashboard_viewed
- report_viewed

### 3. Core Product Actions
- task_created
- task_updated
- task_completed

### 4. Collaboration Actions
- comment_added
- user_mentioned

### 5. Administrative Actions
- user_invited
- role_changed

---

## Why This Taxonomy Matters
A consistent event taxonomy:
- Enables reliable engagement metrics
- Prevents duplicate or ambiguous tracking
- Allows comparison across accounts and time
- Forms the foundation for churn and expansion modeling
