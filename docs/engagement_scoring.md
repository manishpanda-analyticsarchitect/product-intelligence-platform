# Engagement Scoring Framework

## Purpose
The engagement score provides a single, explainable metric that summarizes
overall product usage health at the account level.

The score ranges from 0 to 100.

---

## Scoring Dimensions

### 1. Feature Breadth (0–25)
Measures how many unique features an account actively uses.

Higher score:
- Uses multiple core and advanced features

Lower score:
- Uses only one or two basic features

---

### 2. Feature Depth (0–25)
Measures how intensively features are used.

Higher score:
- High volume of meaningful actions
- Frequent usage across days

Lower score:
- Sporadic or minimal usage

---

### 3. Stickiness (0–25)
Measures how frequently users return to the product.

Higher score:
- Daily or weekly active users

Lower score:
- Monthly or irregular usage

---

### 4. Recency (0–25)
Measures how recently the product was used.

Higher score:
- Activity within last 1–3 days

Lower score:
- No recent activity

---

## Interpreting the Score

| Score Range | Interpretation |
|---|---|
| 75–100 | Healthy |
| 40–74 | Monitor |
| 0–39 | At Risk |

---

## Why This Score Matters
- Enables early churn detection
- Helps prioritize customer success actions
- Allows tracking engagement trends over time
- Provides a common language across teams
